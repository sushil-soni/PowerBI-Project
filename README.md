# 📊 Power BI Projects

This repository contains Power BI dashboards showcasing data visualization, KPI tracking, and business insights through interactive analytics.

## 🟡 Blinkit Sales Dashboard

- Analyzed $1.20M total sales across 8,523 items with avg rating of 3.9
- Sales breakdown by item type — Household, Dairy, Snacks, Fruits & more
- Outlet-wise analysis by size (Small/Medium/High) and location (Tier 1/2/3)
- Interactive filter panel by Outlet Location, Outlet Size & Item Type

**Tools Used:** Power BI, DAX, Power Query
